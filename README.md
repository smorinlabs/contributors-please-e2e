# contributors-please-e2e

Scratch target repository for live `contributors-please-action` adoption tests.

This repository is intentionally mutable. Workflows in
`smorinlabs/contributors-please-test` may create commits, branches, pull
requests, labels, `CONTRIBUTORS.md`, and `.contributors.jsonl` here while
verifying the published Action behaves correctly in a real GitHub repository.

Do not use this repository as product source. Test workflows may reset generated
files and close or replace test pull requests.
